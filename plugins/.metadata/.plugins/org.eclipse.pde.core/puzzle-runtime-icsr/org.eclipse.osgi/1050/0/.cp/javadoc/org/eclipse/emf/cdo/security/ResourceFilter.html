Interface ResourceFilter

  • All Superinterfaces:
    CDOObject, CDOWithID, org.eclipse.emf.ecore.EObject, org.eclipse.emf.common.notify.Notifier, PermissionFilter
    All Known Implementing Classes:
    ResourceFilterImpl

    public interface ResourceFilter
extends PermissionFilter
    A representation of the model object 'Resource Filter'.
    Since:
    4.3

    The following features are supported:

    • Path
    • Pattern Style
    • Folders
    • Text Resources
    • Binary Resources
    • Model Resources
    • Model Objects
    • Include Parents
    • Include Root
